# LovingMyCurves - Design System

## Brand Colors

### Primary
- **Brand Pink**: `#E6007E` / `rgb(230, 0, 126)` - Primary CTA buttons, links, accents
- **Brand Pink Hover**: `rgba(230, 0, 126, 0.83)` - Button hover state

### Neutrals
- **Black**: `#000000` - Primary text
- **Dark Gray**: `#333333` / `rgb(51, 51, 51)` - Secondary text
- **Medium Gray**: `#555555` / `rgb(85, 85, 85)` - Body text
- **Light Gray**: `#888888` / `rgb(136, 136, 136)` - Tertiary text
- **Border Gray**: `#E5E5E5` / `rgb(229, 229, 229)` - Borders, dividers
- **Background Gray**: `#F8F8F8` / `#F9F9F9` - Section backgrounds
- **White**: `#FFFFFF` - Backgrounds, card backgrounds

## Typography

### Font Families
- **Primary (Headings)**: `Montserrat` - Clean, modern sans-serif
- **Secondary (Body)**: `Open Sans` - Readable body text
- **Icons**: `FontAwesome` - Icon font

### Font Sizes (to be extracted from components)
- Headings: Various sizes
- Body: Standard sizes
- CTA buttons: Medium weight

## Spacing
- Standard padding for sections
- Card gaps and margins
- Button padding

## Components

### Buttons
- **Primary CTA**: Pink background (#E6007E), white text, rounded corners
- **Hover**: Slightly transparent pink
- **Text**: Montserrat font

### Cards
- White background
- Box shadows for elevation
- Rounded corners
- Hover effects (subtle lift/shadow)

### Product Cards
- Image container
- Title
- Price in KSh currency format
- Hover: Scale/shadow effect

### Navigation
- Sticky header behavior
- Desktop: Horizontal menu
- Mobile: Hamburger menu

## Layout

### Containers
- Max-width for content areas
- Responsive breakpoints:
  - Desktop: 1440px+
  - Tablet: 768px - 1439px
  - Mobile: < 768px

### Grid Systems
- Category cards: 4-column on desktop → 2-column tablet → 1-column mobile
- Product cards: 3-column on desktop → 2-column tablet → 1-column mobile
- Footer: 4-column on desktop → 2-column tablet → 1-column mobile

## Effects & Interactions

### Hover States
- Cards: Subtle scale transform + shadow increase
- Buttons: Opacity change or background darken
- Links: Color change to brand pink

### Scroll Behaviors
- Header: Sticky positioning
- Possible scroll-triggered animations (to be confirmed)

### Carousel/Slider
- Hero slider: Auto-rotating with slide transitions
- Possibly partner logos carousel

## Icons & Assets
- Social media icons: Facebook, Instagram, Pinterest
- WhatsApp floating button (bottom right)
- Search icon
- Cart icon
- Hamburger menu icon (mobile)

## Currency & Locale
- Currency: KSh (Kenyan Shilling)
- Phone format: +254 XXX XXX XXX
